- **爽约**：无合同之债意图，且无主观过错，无需承担赔偿责任。
    
- **驾车人单独过错**：
    
    - 违章驾驶 → 一般侵权，承担赔偿责任。
        
    - 仅一般过失 → 应当减轻其赔偿责任。
        
    - 故意或重大过失 → 承担全部赔偿责任，无免责。
        
- **免责条款效力**：事先约定造成人身损害免责 → **无效**。
    
- **混合过错**（驾车人 + 第三人）：
    
    - 仅驾车人过错 → 驾车人赔偿。
        
    - 仅第三人过错 → 第三人赔偿。
        
    - 均有过错 → 按份承担责任（无意思联络的数人侵权）。
        
- **搭车人过错**：相应减轻责任人的侵权责任。
    
- **易错点**：免费搭车发生事故一律免责 ❌（正确：故意或重大过失造成的损害不免责）。
